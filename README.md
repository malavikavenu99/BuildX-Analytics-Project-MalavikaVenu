# Campus Recruitment Analytics Dashboard

## Project Overview

This project analyzes the Campus Recruitment dataset using Python, SQL, and Power BI. The objective is to clean the data, answer business questions, and build an interactive dashboard to understand placement trends and salary patterns.

---

## Dataset Selected

Campus Recruitment Dataset (Placement_Data_Full_Class.csv)

---

## Why I Selected This Dataset

I selected this dataset because it provides valuable information about student placements, academic performance, work experience, MBA specialization, and salary. It is suitable for practicing the complete data analytics workflow from data cleaning to dashboard creation.

---

## Business Questions Answered

1. How many students were successfully placed?
2. Which degree type has the highest number of students?
3. Which MBA specialization has the highest average salary?

---

## Tools Used

- Python (Pandas)
- Google Colab
- SQLite
- Power BI Desktop
- GitHub

---

## Key Insight

Students with work experience and strong academic performance generally achieved better placement outcomes and higher salary packages.
